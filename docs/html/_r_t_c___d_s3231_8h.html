void RTC_init ( )

Initializes the RTC (Real-Time Clock) module.

This function attempts to initialize the RTC module. If the initialization fails, it prints an error message to the serial monitor and halts the execution indefinitely.

◆ RTC_power_loss()

void RTC_power_loss ( )

Checks if the RTC (Real-Time Clock) has lost power and adjusts the time if necessary.

This function checks if the RTC has lost power using the lostPower() method. If the RTC has lost power, it prints a message to the serial monitor (for debugging purposes) and sets the RTC to the compile time using the adjust() method with the current date and time.
