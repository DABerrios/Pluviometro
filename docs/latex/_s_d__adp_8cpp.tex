void SD\+\_\+init (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Initializes the SD card and mounts it. 

This function sets up the SPI communication for the SD card, configures the chip select (CS) pin, and attempts to mount the SD card. If the SD card fails to mount, an error message is printed to the Serial monitor.

\begin{DoxyNote}{Note}
Ensure that the SD card is properly connected before calling this function.
\end{DoxyNote}
